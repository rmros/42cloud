## About This Project
This project is a React-based SPA. It aimed at providing a complete Dashboard Portal to the IaaS platform based on OpenStack.

## Frontend Technology stack
- React
- Redux
- React-Router
- Redux-Thunk
- Webpack
- Babel
- CSS-Modules
- Echarts
- Ant-Design
