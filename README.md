## 关于项目
此项目是基于 React 技术栈实现的一个 Web 管理控制台。利用 OpenStack 开源平台提供的后端 API 管理数据中心的计算、存储、网络等资源

## 前端技术栈
- React
- Redux
- React-Router
- Redux-Thunk
- Webpack
- Babel
- CSS-Modules
- Echarts
- Ant-Design
